# 黒檀の刃

## *見よ，この景色を。人間の浅ましさがこの絶望をもたらしたのだ。*

聖霊の加護を失い滅亡した国，アーケイ。「黒檀の誓約」を持つ主人公エス  
は，消息を絶った妹を救うべく魔物の巣と化した都に足を踏み入れる。血と  
狂気が支配する世界に果たして救いはあるのか。
